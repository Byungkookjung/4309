## How to Use

1. Open `index.html` in your web browser
2. Type a task in the input field and click "Add" or press Enter
3. Click the checkbox to mark tasks as complete
4. Double-click a task or click "Edit" to modify it
5. Use the filter buttons to view All, Active, or Completed tasks
6. Click "Delete" to remove a task
7. Click "Clear Completed" to remove all completed tasks at once

## Features in Detail

- **Add Tasks**: Enter a task and press Enter or click Add
- **Complete Tasks**: Click the checkbox next to any task
- **Edit Tasks**: Double-click a task or click the Edit button
- **Delete Tasks**: Click the Delete button on any task
- **Filter**: Use the filter buttons to show All, Active, or Completed tasks
- **Persistent Storage**: All tasks are automatically saved in your browser

## Installation

No installation required! Just open `index.html` in any modern web browser.

Your tasks will be saved automatically in your browser's local storage, so they'll persist even after you close the browser.
